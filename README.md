# Автоматизация диагностики пневмонии с помощью нейронных сетей

## Описание проекта

Проект разработан для **медицинской клиники** с целью автоматизации процесса выявления пневмонии у пациентов. Система состоит из двух этапов диагностики, образующих полный цикл выявления заболевания.

## Бизнес-задачи

### Задача 1: Скрининг по симптомам
Автоматизация первичного скрининга пациента для проверки наличия пневмонии на основе клинических симптомов (табличные данные -> полносвязная нейронная сеть).

### Задача 2: Анализ рентгеновских снимков
Автоматизация проверки рентгеновских снимков грудной клетки пациента для выявления пневмонии (изображения -> свёрточная нейронная сеть).

### Связь задач
Первая задача используется для **первичного скрининга**, вторая - **после прохождения рентгена**. В совокупности они образуют полный цикл выявления пневмонии.

Схема бизнес-процесса: [docs/path_diagram.png](docs/path_diagram.png)

## Обоснование

В период сезонных заболеваний в поликлиники обращается множество пациентов с признаками респираторных заболеваний. Для быстрого выявления тяжёлого заболевания — пневмонии — необходима автоматизация этого процесса.
